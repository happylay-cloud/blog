## 重置全局样式

```css
/* 全局样式表 */
html,
#app,
body {
  height: 100%;
  margin: 0;
  padding: 0;
}
```

## 导入全局样式

```javascript
// 导入全局样式表
import "./assets/css/reset.css";
import ElementUI from "element-ui";
import "element-ui/lib/theme-chalk/index.css";
Vue.use(ElementUI);
```
